# 🚦 Traffic Monitoring System using YOLOv8

A real-time traffic monitoring system built using deep learning and computer vision. This project uses YOLOv8 for vehicle detection and tracking to analyze traffic flow from live video streams such as webcams or CCTV cameras.

---

## 📌 Features

- 🚗 Real-time vehicle detection  
- 🔍 Object tracking with unique IDs  
- 📊 Vehicle counting system  
- 🎥 Supports webcam, video files, and IP cameras  
- ⚡ Fast and efficient (real-time performance)  
- 🧠 Built using state-of-the-art deep learning model  

---

## 🛠️ Tech Stack

- **Language:** Python  
- **Libraries & Frameworks:**
  - YOLOv8 (Object Detection & Tracking)
  - OpenCV (Video Processing)
  - PyTorch (Deep Learning Backend)

---

## 🏗️ System Architecture

Video Input → Frame Extraction → YOLOv8 Detection → Object Tracking → Vehicle Counting → Display Output
